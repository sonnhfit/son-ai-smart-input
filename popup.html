Smart Input Assistant

How to use this extension:

  1. Select text in any input field or focus on an input
  2. The prompt popup will appear automatically
  3. Alternatively, press Ctrl+I (or Cmd+I on Mac) to open the popup
  4. Enter your desired input in the popup
  5. Click Submit to process and update the text

Cách sử dụng extension:

  1. Chọn văn bản trong bất kỳ trường nhập liệu nào hoặc focus vào một trường nhập liệu
  2. Cửa sổ popup sẽ tự động xuất hiện
  3. Hoặc, nhấn Ctrl+I (hoặc Cmd+I trên Mac) để mở popup
  4. Nhập nội dung mong muốn vào popup
  5. Nhấp vào Gửi để xử lý và cập nhật văn bản
